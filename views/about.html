About

Find out more about the Starfield API project, how & why it started and some additional information.

History & Background

The project was created out of passion for Starfield by fans of Bethesda games. The project aims to make the lore & information from the game available to all in easy to use & consume formats.

Starfield API is a REST API built using Go (Golang), Fiber & Postgres and is hosted on Railway. The goal is to create a simple, stable & robust service for developers and fans to use as they see fit.
