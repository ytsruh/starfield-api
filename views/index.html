Welcome to the Starfield API

The community resource for all things Starfield. Access a variety of data from across the Settled Systems in easy to consume formats.

Get Started

To get started you must have an API key. To generate an API key, register for an account and create one inside the dashboard.

Upto 2 API keys can be created and deleted for security and easy management.

Documentation

After you've created an API key you can visit our documentation to find the available endpoints and base url. Your api key can be included in either the request headers or as a query string.
