# Starfield API

Public Rest API to make date from the Starfield game.

## Built using

- [GoLang](https://go.dev/)
- [Fiber](https://gofiber.io/)
- [Gorm](https://gorm.io/)
- [Postgres](https://www.postgresql.org/)
- [HTMX](https://htmx.org/)
- [Tailwind](https://tailwindcss.com/)
- [RippleUI](https://www.ripple-ui.com/)
- [css.gg Icons](https://css.gg/)
